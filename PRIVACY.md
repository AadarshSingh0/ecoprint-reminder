# Privacy Policy for EcoPrint Reminder

EcoPrint Reminder does not collect, sell, transmit, or share personal data.

The extension stores print-footprint statistics locally in the user's browser using Chrome's local extension storage. This may include:

- estimated printed page counts
- estimated tree-equivalent values
- avoided-print counts
- pages saved
- estimated trees saved
- title/URL of the most recent counted or avoided print event

This data remains on the user's device and is not sent to any external server.

EcoPrint Reminder does not use analytics, advertising, tracking cookies, or third-party data collection.

Users can reset all stored data from the extension popup at any time.
